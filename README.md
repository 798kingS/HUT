# 湖院通 - 校园生活服务平台

## 项目简介

湖院通是一个专为湖州学院学生设计的微信小程序，提供校园拼车、二手书籍交易和校园论坛三大核心功能，旨在为校园生活提供便利。

## 功能特性

### 🚗 校园拼车
- **发布拼车信息**：支持发布出发地、目的地、时间、车型、座位数、价格等信息
- **拼车列表**：浏览所有拼车信息，支持筛选和搜索
- **互动功能**：点赞、收藏、分享、评论
- **图片上传**：支持上传车辆照片

### 📚 校园卖书
- **发布书籍信息**：包含书名、作者、出版社、ISBN、价格等详细信息
- **书籍列表**：浏览所有二手书籍，支持分类查看
- **收藏功能**：收藏感兴趣的书籍
- **图片展示**：支持多张书籍照片

### 💬 校园论坛
- **发布帖子**：支持文字、图片内容发布
- **论坛列表**：浏览所有讨论帖子
- **互动功能**：点赞、评论、收藏、分享
- **话题讨论**：校园生活、学习交流、活动分享

### 👤 个人中心
- **用户信息**：昵称、头像、性别、生日、联系方式等
- **我的发布**：查看自己发布的所有内容
- **我的收藏**：管理收藏的内容
- **设置功能**：退出登录、清除缓存、关于我们

## 技术架构

- **前端框架**：微信小程序原生开发
- **样式设计**：WXSS + 响应式布局
- **数据存储**：微信云开发 / 自建后端API
- **图片上传**：微信云存储
- **用户认证**：微信登录

## 开发环境

- 微信开发者工具
- Node.js (可选，用于后端开发)
- 微信小程序基础库 2.19.4+

## 安装和运行

1. **克隆项目**
   ```bash
   git clone [项目地址]
   cd 湖院通
   ```

2. **打开微信开发者工具**
   - 导入项目
   - 选择小程序项目
   - 填入你的AppID

3. **配置项目**
   - 修改 `project.config.json` 中的 `appid`
   - 配置后端API地址（在 `app.js` 中修改 `baseUrl`）

4. **运行项目**
   - 在微信开发者工具中点击"编译"
   - 在模拟器中预览效果

## 配置说明

### 后端API配置
在 `app.js` 中修改 `baseUrl` 为你的后端服务地址：
```javascript
globalData: {
  baseUrl: 'https://your-api-domain.com/api'
}
```

### 微信登录配置
确保在微信公众平台配置了正确的域名和服务器地址。
